# 캠퍼스 미연시 INFO API

## 사용법

```
![](https://your-url.railway.app/api/campus_status?loc=중앙+광장&t=03월02일+09:30&p=강하율.20.60.카페테리아,이서연.20.60.강의실,최유나.20.60.편의점)
```

## 파라미터

| 파라미터 | 설명 | 예시 |
|---|---|---|
| loc | 현재 장소 | 중앙+광장 |
| t | 날짜/시간 | 03월02일+09:30 |
| p | 히로인 데이터 (이름.호감도.경계도.위치) | 강하율.20.60.카페테리아 |

## 히로인 위치값
- 강의실
- 카페테리아
- 편의점
- 기숙사
- 공원
